# Arduino Car
This is a mini project to familiar with the Arduino development kit.

Main Components: 
1. Arduino UNO
2. HC-05 Bluetooth Module
3. Motors & Wheels
4. L298N Motoe Driver

The file test.apk is created by the MIT App Inventor. The app could be installed in Android phone to control the movement of the Arduino Car.

The file motor_control.ino is used to control the motor speed by using PWM. This program would be loaded to the Arduino UNO.
